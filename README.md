# 💰 Money Manager – Income & Expense Tracking Application

## 📌 Project Overview

The **Money Manager – Income & Expense Tracking Application** is a full-stack web application designed to help users efficiently manage their personal finances. It allows users to track income, record expenses, monitor spending habits, and maintain financial balance in a simple and user-friendly way.

---

## 🚀 Features

* ➕ Add, update, and delete income records
* ➖ Add, update, and delete expense records
* 📊 View total income, total expenses, and remaining balance
* 📅 Track transactions with date and category
* 🔍 Filter and search transactions
* 📈 Dashboard for financial overview
* 🔐 Secure backend with REST APIs

---

## 🏗️ System Architecture

* Frontend (React) communicates with backend using REST APIs
* Backend (Spring Boot) handles business logic and data processing
* MySQL database stores user transactions and financial data

## ⚙️ Setup Instructions

### 🔹 Backend (Spring Boot)

1. Clone the repository
2. Open project in IDE (IntelliJ / Eclipse)
3. Configure MySQL database in `application.properties`
4. Run the Spring Boot application

### 🔹 Frontend (React)

1. Navigate to frontend folder
2. Install dependencies:

   ```
   npm install
   ```
3. Start the application:

   ```
   npm run dev
   ```

---

## 🔐 Security Note

Sensitive information like API keys and passwords are not stored directly in the code. Environment variables are used to ensure secure configuration.
